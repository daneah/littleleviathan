# littleleviathan

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Run your tests
```
yarn test:e2e
```

### Lints and fixes files
```
yarn lint
```

### Formats files
```
yarn format
```

### Run your unit tests
```
yarn test:unit
```
